## Описание модулей

### Модуль 1: `unit.py`

Абстрактный базовый класс `Unit`. Содержит:
- Шесть характеристик (`strength`, `dexterity`, `constitution`, `wisdom`, `intelligence`, `charisma`)
- Абстрактные методы:
  - `calculate_max_health()` — максимальное здоровье
  - `calculate_damage()` — базовый урон
  - `calculate_defense()` — защита

**Импорты:** `from abc import ABC, abstractmethod`, `import math`

---

### Модуль 2: `character.py`

Класс `Character`, наследующий `Unit`. Особенности:
- Параметр `character_class` (`'warrior'`, `'mage'`, `'hunter'`)
- Ветвление логики внутри методов:
  - `calculate_damage()`: разные формулы для каждого класса
  - `calculate_defense()`: разные формулы для каждого класса
  - `calculate_max_health()`: единая формула для всех
- В конструкторе сразу вычисляются и сохраняются `max_health`, `current_health`, `damage`, `defense`
- Валидация класса: при неверном значении выбрасывается `ValueError`

**Формулы:**

| Класс | Урон | Защита |
|-------|------|--------|
| Воин | сила × 2,2 + телосложение ÷ 3 | телосложение × 1,8 + сила ÷ 4 |
| Маг | интеллект × 2,5 + мудрость ÷ 2 | мудрость × 1,3 + интеллект ÷ 6 |
| Охотник | ловкость × 1,9 + сила ÷ 3 | ловкость × 1,6 + телосложение ÷ 5 |

**Здоровье для всех:** телосложение × 10 + сила ÷ 2

---

### Модуль 3: `monster.py`

Класс `Monster`, наследующий `Unit`. Особенности:
- Конструктор вычисляет и сохраняет `max_health`, `current_health`, `damage`, `defense`
- Метод `calculate_max_mana()` возвращает 0 (монстры не используют магию)

**Формулы монстра:**
- Здоровье: телосложение × 8 + сила ÷ 3
- Урон: сила × 2 + телосложение ÷ 5
- Защита: телосложение × 1,2 + сила ÷ 5


### 'spells.py'

Содержит:
- Абстрактный класс `Spell` с атрибутами `name`, `damage`, `mana_cost` и абстрактным методом `cast()`
- Три конкретных заклинания:
  - `Fireball` — урон 35, стоимость 15 маны
  - `IceLance` — урон 25, стоимость 10 маны
  - `LightningBolt` — урон 40, стоимость 20 маны


### 'demo.py'

Демонстрация работы всех классов:
- Создание персонажей (воин, маг, охотник) и монстра
- Добавление заклинаний
- Применение заклинаний с проверкой маны
- Вывод характеристик в консоль


## Система маны и заклинаний (расширение)

В `unit.py` добавлены:
- Атрибуты: `spells` (список заклинаний), `mana` (текущая мана)
- Абстрактный метод `calculate_max_mana()`
- Метод `add_spell(spell)` — добавляет заклинание
- Метод `cast_spell(index)` — применяет заклинание с проверкой маны

В `character.py` реализован метод `calculate_max_mana()`:

| Класс | Формула максимальной маны |
|-------|--------------------------|
| Воин | интеллект + сила ÷ 2 |
| Маг | интеллект × 3 + мудрость |
| Охотник | ловкость × 1,5 + мудрость ÷ 2 |
